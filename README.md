# 🚀 Nik.workspace - Professional Smart Editor

**NIK.workspace** is a futuristic, browser-based writing environment designed for creators and engineers. It combines a clean **Cyberpunk UI** with powerful utility features like folder organization and live data persistence.

Developed by **Nikhil Misal**.

---

## ✨ Key Features

- 📁 **Folder Management:** Organize your thoughts and logs into custom folders.
- ⚡ **Auto-Correction:** Built-in dictionary to fix common spelling mistakes instantly (e.g., *teh* → *the*).
- 💾 **Local Persistence:** Data is automatically saved to your browser’s `localStorage`. No data loss on refresh!
- 📊 **Live Analytics:** Track **Characters**, **Words**, and **Alphabets** in real-time.
- 📱 **Fully Responsive:** Works perfectly on Desktop, Tablet, and Mobile.
- 🕒 **System Sync:** Integrated Real-time Clock and Date display.
- 🎨 **Animated Branding:** A premium, pulsing neon-gradient interface.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Custom Glassmorphism, Neon UI, and Keyframe Animations.
- **JavaScript (ES6):** Real-time data engine, folder logic, and autocorrect system.
